## **📝 개요**
> **프로젝트** : 스마트 파인더
>
> **인원** : 4인
> 
> **주제** : Kakao api의 stt를 활용하여 음성을 통한 수납함 위치 알림
> 
> **제작 기간** : 2022.3 ~ 2022.06
> 
> **주요 기능**   
> > * kakao api를 활용한 음성인식 데이터 처리
> > * 음성데이터를 활용한 하드웨어 제어
> > * 간단한 구조의 어플을 활용한 하드웨어 제어
> **참고 포트폴리오**
